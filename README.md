## Giới thiệu

#Đây là một dự án mô phỏng hệ điều hành đơn giản, dự án tập trung vào các thành phần cốt lõi của một OS, bao gồm:

Quản lý tiến trình (Process Management)

Bộ lập lịch (Scheduler – MLQ)

Quản lý bộ nhớ (Memory Management – Paging)

Hệ thống file cơ bản (File System)

System Call để giao tiếp giữa tiến trình và OS
